Step 1. Create repository
Step 2. Open Terminal
Step 3. Type pwd which is present working directory
Step 4. Initialize the git, type git init.  
Step 5. Check the list, type ls.
Step 6. Add the file to staging file type git add .
Step 7. See the file, type git status.
Step 8. Commit the file fro local repository, type git commit -m 'message here'.
Step 9. Paste the remote URL then push
